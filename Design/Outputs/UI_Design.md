# 《万妖录：躺平修仙》 - UI 设计文档 (UI Design)

## 1. UI 总体设计原则
*   **交互模式**：针对移动端 (iOS/Android) 优化，主打竖屏单手操作。
*   **设计风格**：贴合“搞怪修仙”、“极简剪纸”的美术定位。界面轻量、直观，避免过度复杂的层级设计。
*   **反馈优先级**：强调“多巴胺反馈”，抽卡（抓捕）、升级、变异等环节必须有强烈的动效和视觉中心凸显。

## 2. 局内战斗 UI (In-Game HUD)

### 2.1 核心状态显示
*   **血条 (HP) 与 经验条 (灵气槽)**：置于屏幕上方或角色头顶，直观显示当前状态与升级进度。
*   **倒计时/时间轴**：顶部显示单局存活时间（10-15分钟上限）。
*   **满屏伤害数字**：要求数字跳动具有夸张的物理弹跳感，暴击伤害需有放大和变色（如红色/金色）强调。

### 2.2 操作与技能指示
*   **单摇杆**：屏幕左下角区域（或随手指按压位置动态生成虚拟摇杆）。
*   **抛葫芦按钮**：右下角极具辨识度的大按钮，具备明显的冷却 (CD) 转圈提示与可用状态高亮。
*   **画面预警提示**：屏幕边缘需随机刷出“带感叹号”的方向指示器，用于引导玩家追击稀有妖兽/变异妖兽。
*   **BOSS 状态栏**：大体积血条，包含阶段标记。当BOSS血量进入“破防虚弱状态”时，血条UI需闪烁或高亮，提示玩家“快扔葫芦”。

### 2.3 局内升级与流派构筑
*   **3选1卡牌面板**：
    *   升级时游戏暂停/背景压暗，弹出三张技能卡牌。
    *   卡牌需通过边框颜色或底纹图标清晰区分流派（Tag：🟢体修、🔵法修、🟠御兽）。
    *   卡牌展示：技能名称、图标、简短搞怪描述、当前等级。
    *   面板底部或侧边提供显眼的“看广告刷新”按钮。

## 3. 局外系统 UI (Out-of-Game UI)

### 3.1 洞府大本营 (主界面)
*   **背景**：动态展示玩家放置在洞府打黑工的妖兽（种地、挖矿）。
*   **放置收益UI**：中心或宝箱位置显示离线累积的灵石、材料，点击后有大范围物资飞入背包的爽快特效。
*   **主要系统入口**：孵化/盲盒、宠物阵容、天赋境界、装备法宝、商店/通行证。

### 3.2 宠物与图鉴系统
*   **妖兽蛋孵化界面**：
    *   盲盒体验核心界面。包含“普通孵化”与“仙露孵化（保底）”按钮（附带看广告加速选项）。
    *   孵化过程需有倒计时及开盒强光特效，若触发变异，需全屏变暗、特定高光、震动及夸张文字（如“变异大成功！”）。
*   **出战配置面板**：
    *   最多 5 个槽位（随境界解锁）。
    *   界面需清晰显示当前激活的羁绊（五行羁绊、种族羁绊），凑齐神兽【四象大阵】时需有专属高亮发光提示。
*   **图鉴墙**：
    *   卡牌式或卷轴式排列。已收集/未收集状态区分明显。
    *   专门的变异神宠陈列区，满足收集癖。
    *   顶部或底部显示当前图鉴解锁提供的全局属性加成总览。

### 3.3 养成与境界系统
*   **灵根天赋树**：以“修仙经脉”或“星图”形式展现，分支包含：摸鱼心法、肝帝克星、天命之子。
*   **境界突破界面**：展示从“练气”到“渡劫”的进度条，达到瓶颈时显示所需的“突破丹”数量，并有极具仪式感的“历劫”按钮。

### 3.4 装备 (仙器法宝) 界面
*   **穿戴面板**：角色模型居中，四周环绕 4 个装备槽位（主武器、法衣、配饰、祖传葫芦）。
*   **信息展示**：选中法宝时，侧边栏显示基础属性变化及核心机制特效（如吸血、增加抓捕率等）。

### 3.5 商业化 UI
*   **特权与广告**：免广告特权卡（月卡）悬浮购买入口；随处可见但融入UI的广告激励入口（复活、刷新、加速）。
*   **修仙通行证 (Battle Pass)**：长条形等级奖励轨道，分为免费线与付费线，高亮展示当前可领取的极品葫芦或变异蛋奖励。
