# 游戏概念设计框架 (GDD 概览)
**项目代号：** 《万妖录：躺平修仙》
**游戏类型：** 轻度休闲 Roguelite (割草) + 宠物收集养成 (RPG)
**目标平台：** 移动端 (iOS/Android)，主打竖屏单手操作。
**核心卖点：** 极低上手门槛、高随机性多巴胺反馈（变异盲盒）、丰富且夸张的修仙流派爽感。

---

## 1. 核心玩法循环 (Core Loop)

游戏的设计必须让玩家形成“停不下来”的循环体验。

*   **局内体验 (战斗与构筑)：**
    单摇杆控制走位（自动攻击） $\rightarrow$ 击杀妖兽掉落“灵气”与“未孵化的妖兽蛋” $\rightarrow$ 吸收灵气升级，触发**随机3选1技能** $\rightarrow$ 构筑本局流派 $\rightarrow$ 遇到精英怪/BOSS，残血时扔葫芦**拼运气抓捕** $\rightarrow$ 结算通关/死亡。
*   **局外体验 (养成与收集)：**
    带出局内的金币/妖兽蛋 $\rightarrow$ 孵化妖兽蛋（鉴定是否**变异**） $\rightarrow$ 升级主角基础境界（点天赋树） $\rightarrow$ 配置出战宠物阵容 $\rightarrow$ 再次进入关卡。

---

## 2. 核心系统详细设计

### 2.1 宠物抓捕与收集系统（“运气”核心点）
这是游戏最核心的“多巴胺”来源，必须将随机性做到极致：
*   **偶遇机制：** 割草过程中，画面边缘会随机刷出“稀有妖兽”头上带感叹号，限时击杀或抓捕。
*   **葫芦抓捕（拼概率）：** 怪物血量越低，抓捕成功率越高。玩家点击右下角“抛葫芦”技能（有冷却）。抓捕会有类似精灵宝可梦的“晃动三次”动画，把期待感拉满。
*   **变异鉴定（盲盒体验）：** 抓到的宠物或掉落的蛋，在局外孵化时有**极小概率（如 1%~5%）触发变异**。
    *   *普通青蛇 $\rightarrow$ 技能：吐毒液。*
    *   *变异·吞天蟒 $\rightarrow$ 外观变大泛金光，技能：吞噬全屏低级小怪。*
*   **图鉴收集：** 点亮图鉴不仅满足收集癖，还能为全员提供永久的基础属性加成（攻击+1%，血量+1%）。

### 2.2 局内升级与流派构筑 (Build系统)
升级时出现的3选1卡牌，按照流派（Tag）进行分类。玩家凑齐特定数量的同流派卡牌，会触发“流派共鸣（终极技）”。

#### 🟢 体修流（肉身成圣）
*   **定位：** 极度安逸，适合真正“躺平”的玩家。
*   **核心词条：** 气血上限、每秒回血、百分比反伤、霸体。
*   **专属技能：**
    *   *【金钟罩】：* 身边生成一口大钟，怪物撞上去直接粉碎。
    *   *【莽牛劲】：* 没有任何主动远程技能，但每次受击有概率向周围打出恐怖的“反震冲击波”。
*   **流派共鸣（满级）：** **【法天象地】** - 主角体型瞬间变大3倍，一脚踩死一片怪，无视任何控制。

#### 🔵 法修流（万法归一）
*   **定位：** 视觉最华丽，全屏清怪，追求极致DPS。
*   **核心词条：** 技能冷却、投射物数量、法术范围、穿透。
*   **专属技能：**
    *   *【万剑诀】：* 每秒射出追踪飞剑。
    *   *【五雷轰顶】：* 随机在屏幕内劈下闪电，击杀敌人后分裂。
*   **流派共鸣（满级）：** **【诛仙剑阵】** - 屏幕周围生成结界，进入结界的怪物每秒受到高频毁灭打击，视觉上是满屏数字乱飞。

#### 🟠 御兽流（万兽之主）
*   **定位：** 策略性强，依赖局外收集的宠物，享受“群殴”快感。
*   **核心词条：** 召唤物数量、宠物攻速、宠物暴击、主角隐匿（降低怪物仇恨）。
*   **专属技能：**
    *   *【撒豆成兵】：* 召唤一群临时的小树妖向前冲锋。
    *   *【兽血沸腾】：* 让所有场上的宠物进入狂暴状态，体型变大，攻速翻倍。
*   **流派共鸣（满级）：** **【百兽夜行】** - 吹响号角，从屏幕边缘冲出上百只各类妖兽，像海啸一样碾过全图。

## 3. 美术与性能优化规范（低配友好）

*   **2D 骨骼动画 (Spine/Live2D)：** 坚决避免使用复杂的3D模型。所有怪物和主角均采用2D骨骼动画，甚至是只有几个帧的“纸片人”动画，节约内存。
*   **风格化遮百丑：** 建议采用“搞怪修仙表情包风”或“极简中国剪纸风”。不需要精细的光影，用高对比度的颜色区分敌我。
*   **特效池化技术 (Object Pooling)：** 因为后期同屏可能有几百个怪物和飞剑，必须在程序架构上使用对象池，并利用 GPU Instancing 技术来渲染海量怪物，确保在千元机上也能满帧运行不发烫。
*   **视觉爽感代替精度：** 怪物死亡不需要复杂的倒地动画，直接“砰”的一声变成一缕白烟或金币飞向玩家，配合屏幕微震，手感就出来了。

## 4. 商业化与留存设计建议

虽然是休闲游戏，但需要有健康的盈利点，主要以 **混合变现 (IAA + IAP)** 为主：
1.  **广告变现 (IAA)：** 
    *   局内：看广告复活一次；看广告刷新一次“3选1”技能（极其诱人）。
    *   局外：看广告加速妖兽蛋孵化；看广告增加一次高级抓捕葫芦的使用次数。
2.  **内购变现 (IAP)：** 
    *   **免广告特权卡 (月卡)：** 最核心的收入来源，买了之后所有广告奖励直接领。
    *   **特殊宠物皮肤/异色版：** 卖外观，不卖数值，保证普通玩家靠运气也能抓到神兽。
    *   **修仙通行证 (Battle Pass)：** 随着闯关进度，解锁大量的培养材料和极品葫芦。
